## Developer Mono wiring note
The jsonresume.org registry imports the theme package under the name `jsonresume-theme-developer-mono` in `apps/registry/lib/formatters/template/themeConfig.js`.

## Project intent
The end state should be your own combined implementation in this repo, not a blind copy. Use upstream ideas and packages where licenses allow, preserve attribution, and keep your resume content source-first and agent-friendly.
